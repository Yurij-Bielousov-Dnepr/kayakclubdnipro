Н1: Головні підрозділи на сторінці можна виділити більшим шрифтом, наприклад, 24pt, і більш темним кольором, наприклад, #000000.
CSS
h1 {
  font-size: 24pt;
  color: #000000;
}
Н2: Внутрішні заголовки на сторінці можна виділити меншим шрифтом, наприклад, 18pt, і світлим кольором, наприклад, #666666.
CSS
h2 {
  font-size: 18pt;
  color: #666666;
}
Н3: Текст підроздівів на сторінці можна виділити середнім шрифтом, наприклад, 16pt, і нейтральним кольором, наприклад, #888888.
CSS
h3 {
  font-size: 16pt;
  color: #888888;
}
Н4: Текст списків на сторінці можна виділити меншим шрифтом, наприклад, 14pt, і світлим кольором, наприклад, #999999.
CSS
h4 {
  font-size: 14pt;
  color: #999999;
}
Головні підрозділи на сторінці (додаткові стилі)
CSS
h5 {
  font-size: 12pt;
  color: #333333;
  background-color: #CCCCCC;
  border: 1px solid #CCCCCC;
}
Н6: Схований текст на сторінці можна виділити дуже дрібним шрифтом, наприклад, 10pt, і світлим кольором, наприклад, #CCCCCC.
CSS
h6 {
  font-size: 10pt;
  color: #CCCCCC;
}
Р: Параграфи можна відокремити відступами зверху та знизу, наприклад, 16px.
CSS
p {
  margin: 16px 0;
}
Рис: Картинки на сторінці можна обрізати до певного розміру, наприклад, 600x400 пікселів.
CSS
img {
  width: 600px;
  height: 400px;
}
Рк: Перебивки-картинки можна зробити більш привабливими, додавши до них текст або кнопку.
CSS
.rk {
  text-align: center;
  margin-bottom: 20px;
}

.rk > img {
  width: 100%;
}

.rk > p {
  margin-top: 10px;
}

.rk > button {
  margin-top: 10px;
}
